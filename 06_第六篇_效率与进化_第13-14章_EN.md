### Chapter 13: Actions & Batch — The Power of Automation

#### 13.1 The Logic of Recording Actions: What Can and Cannot Be Recorded?

Actions are the core feature for implementing automation in Photoshop. Think of them as "macro commands" — they can record a series of operational steps (such as adjusting layers, applying filters, saving files, etc.) and then replay those steps on other files with a single click.

The Actions system is one of the most powerful automation tools in Photoshop. It can dramatically boost productivity, especially for tasks that need to be performed repeatedly.

##### Recordable Operations

Actions can record the vast majority of operations performed within the Photoshop interface, including but not limited to:

- Executing menu commands (e.g., Filters, Adjustments)
- Layer panel operations (creating layers, adjusting opacity, blend modes)
- Toolbox tool usage (e.g., Marquee tool, Brush tool)
- Dialog parameter settings (e.g., Curves adjustment, Gaussian Blur radius)
- File saving and exporting

Essentially, any operation performed through menus, panels, or dialog boxes can be recorded.

##### Non-Recordable Operations

Actions cannot record operations that require manual, non-programmatic interaction, such as:

- Hand-painting with the Brush tool (unless using a preset brush and recording a keyboard shortcut)
- Manually adjusting anchor points with the Pen tool
- Manually selecting areas with the Magic Wand tool (recording AI-based selection tools like "Select Subject" is possible)
- Any operation that requires you to drag, click, and determine positions on the canvas with the mouse

Understanding these limitations is crucial. Before recording an action, you need to rehearse the workflow in your mind, ensuring every step is recordable.

**Recording Mindset**: Before recording an action, plan out the steps. Make sure each step is clear and repeatable. Avoid steps that require manual judgment.

#### 13.2 Batch Processing: How to Process Hundreds of Images in One Click

Batch processing is automation built on top of Actions. It allows you to apply the same Action to hundreds or even thousands of images at once, achieving fully automated bulk processing.

##### Core Steps for Setting Up Batch Processing

1. **Prepare the Action**: First, record the Action you need to apply (e.g., adjusting exposure, sharpening, saving to a specific format).

2. **Open the Batch Dialog**: Go to `File > Automate > Batch`.

3. **Set the Source Folder**: Select the folder containing the images to process.

4. **Specify the Action**: In the "Action" dropdown menu, select the Action you recorded.

5. **Set the Destination Folder**: Choose the save location for processed images (can be the same as the source folder, but it's recommended to specify a separate output folder to avoid overwriting originals).

6. **Set File Naming**: If saving as new files, you can specify naming conventions.

7. **Run the Batch**: Click "OK." Photoshop will automatically open each image in the folder, apply the Action one by one, and save the processed files.

##### Advanced Tip: Conditional Mode Change

In batch processing, you can also add a "Conditional Mode Change" Action to automatically convert source files from different formats (e.g., JPG, PNG, TIFF) to the same color mode (e.g., RGB), ensuring subsequent Actions execute correctly.

#### 13.3 Droplets: Creating Desktop Shortcut Processors

Droplets are an even more convenient form of automation than batch processing. They are standalone, executable shortcuts that can be placed directly on your desktop.

##### Creating and Using Droplets

1. **Record an Action**: Again, you need to prepare the Action to be applied first.

2. **Create the Droplet**: Go to `File > Automate > Create Droplet`.

3. **Set Options**: In the dialog, select the Action to use and set options such as the output folder.

4. **Save the Droplet**: Save the created file to your desktop.

5. **Use the Droplet**: Simply drag and drop the image or folder you want to process onto the Droplet icon on your desktop. Photoshop will automatically launch and apply the Action.

**Advantage**: No need to open Photoshop first — complete batch processing directly through a desktop shortcut. The operation is more intuitive and efficient. Ideal for users who frequently need to repeat the same processing workflow.

- **Folder Categorization**: Create a clear folder structure based on asset type (images, textures, fonts, vectors, brushes, etc.). For example: `Asset Library > Images > People`, `Asset Library > Textures > Wood Grain`.

- **Cloud Backup**: Sync your asset library to cloud storage (e.g., Google Drive, Dropbox, Alibaba Cloud Drive, etc.) to prevent data loss. Local storage always carries the risk of failure.

#### 14.2 Open Source & Alternatives: An Introduction to PhotoGIMP and GIMP

While Photoshop is the industry standard, it is not the only option. Familiarizing yourself with open-source and free alternatives allows you to choose the best tool for different scenarios, while understanding that core image-editing thinking is not tied to any specific software.

##### GIMP: A Free and Open-Source Photoshop Alternative

- **Overview**: GIMP (GNU Image Manipulation Program) is a powerful, free, and open-source image editor. It can handle the vast majority of Photoshop's features, including layers, masks, filters, painting, and more.

- **Advantages**: Completely free, available on Windows, macOS, Linux, and other platforms. Has an active community and a rich selection of plugins. Very friendly for individual users with limited budgets.

- **Disadvantages**: The interface and workflow logic differ from Photoshop, resulting in a steeper learning curve. Some professional features (e.g., Camera Raw-style RAW processing, advanced AI tools) are not as mature as Photoshop's. The ecosystem and plugin quality can be inconsistent.

##### PhotoGIMP: Making GIMP Feel Like Photoshop

- **Overview**: PhotoGIMP is a pre-configuration set for GIMP that modifies GIMP's interface, keyboard shortcuts, menu layout, and more to closely resemble Photoshop's style, significantly reducing the learning curve for users transitioning from Photoshop to GIMP.

- **Advantages**: Use familiar Photoshop workflows while enjoying GIMP's free and open-source nature. Lower learning costs.

**Core Insight**: Regardless of the software used, the core image-editing mindset (layers, masks, selections, color grading logic) is universal. Master these underlying principles, and you can quickly adapt to any image editing software.

#### 14.3 Deliberate Practice: How to Deconstruct Master Works and Practice Through Imitation

Improving your Photoshop skills requires extensive practice. The most effective form of practice is not aimless drilling of basic exercises, but "deliberate practice" — purposefully deconstructing and imitating master works.

The core of deliberate practice lies in a cycle of purpose, feedback, and improvement. It differs from simple repetition — you must constantly push the boundaries of your ability.

##### Steps to Deconstruct a Master's Work

1. **Observation and Analysis**:
   - **Composition**: Analyze the layout, subject placement, and visual guidance in the image. Note the distribution and relationships of elements within the frame.
   - **Light and Shadow**: Determine the light source direction, the distribution of light and dark areas, and contrast settings. Observe whether shadows have soft or hard edges and how highlights shape the form of objects. Note how light and shadow guide the viewer's eye.
   - **Color**: Analyze the overall color tone, color scheme (e.g., complementary colors, analogous colors), saturation distribution, and whether localized color grading was used. Understand how color conveys emotion.
   - **Techniques**: Guess which techniques were used (e.g., double exposure, compositing, filters, brushes, etc.). Infer the production process and tools used.

2. **Layer-by-Layer Reconstruction**:
   - **New Document**: Create a new document with the same dimensions and resolution as the master work. Ensure technical specifications match.
   - **Build in Layers**: Like building with blocks, use layers, masks, adjustment layers, and other tools to reconstruct the work layer by layer. Analyze the structure and purpose of each layer.
   - **Record Steps**: During reconstruction, document the method and parameters for each effect. This process itself is an excellent learning exercise. Recording helps organize your thoughts.
   - **Compare and Adjust**: Constantly compare with the original image, adjusting parameters and effects to get as close as possible. Repeated comparison and adjustment are key to progress.

##### Advanced Imitation Practice: Creative Mimicry

- **Modify Parameters**: After completing an imitation, try changing certain parameters (e.g., color tone, light direction) and observe the effect, deepening your understanding of the tools. Understand the impact by altering parameters.

- **Swap Elements**: Replace the assets in the master work with your own, completing a "re-creation." Reproducing a master's effect with your own assets is excellent practice.

- **Blend Styles**: Try combining the styles of different masters to create your own unique style. Style fusion is the starting point of innovation.

**Practice Mindset**: The goal of imitation is not to copy, but to understand. Through deconstruction and imitation, you can grasp how a master thinks and how they use tools to realize creative ideas. The progress rate from this learning approach is far faster than learning tool usage in isolation.

### Chapter 15: Color Management — Precise Delivery from Screen to Print

#### 15.1 Color Space and Color Mode: Preventing Color Shifts at the Source

Color management is the foundation of professional image processing. If your color management workflow is chaotic, the colors you see on screen will be completely different from what is printed or what appears on a phone.

Color management is the bedrock of professional work. Without proper color management, even the best technique cannot guarantee output quality.

##### Color Space

A color space defines the "range" or "container" of colors in the digital world. Different color spaces contain different numbers of colors.

- **sRGB**: The most universal color space. Almost all web browsers, phones, and TVs use sRGB by default. If your primary purpose for editing images is online sharing, always use sRGB. sRGB is the internet standard, ensuring compatibility.
- **Adobe RGB (1998)**: Has a wider gamut than sRGB, especially capturing more greens and cyans. Primarily used for professional photography and high-end printing. Suitable for professional workflows requiring a wide gamut.
- **ProPhoto RGB**: Has the widest gamut, even including some colors invisible to the human eye. Mainly used as an intermediate working space for professional RAW processing in photography. It must ultimately be converted to sRGB or Adobe RGB for output. Extremely wide gamut, suitable as an intermediate format.

**Workflow Recommendation**: In Photoshop, set your working color space to Adobe RGB (if you have a professional monitor) or sRGB (if your monitor does not support a wide gamut). When processing RAW files in Camera Raw, you can set the output color space in the "Workflow Options."

##### Color Mode

- **RGB**: An additive color model based on light. Used for screen display (computers, phones, TVs). Photoshop's default and most commonly used color mode.
- **CMYK**: A subtractive color model based on ink. Used for print publishing. Images edited in RGB mode must be converted to CMYK before printing, or severe color shifts will occur. Printing uses subtractive color mixing, which is fundamentally different from screen display.
- **Grayscale**: Black-and-white mode. Used for black-and-white printing or artistic creation. Uses only black, white, and gray, suitable for specific artistic effects.

**Conversion Mindset**: Always stay in RGB mode during editing. Only convert to CMYK mode when you have confirmed the image will be printed, and use `View > Proof Colors` to simulate the print effect.

#### 15.2 Monitor Calibration: Let Your Eyes See the Truth

An uncalibrated monitor is like a ruler with inaccurate markings — no matter how skilled you are at image editing, you cannot make accurate judgments.

Monitor calibration is foundational to professional work. Without an accurate monitor, all adjustments are blind.

- **Hardware Calibration**: Use a professional colorimeter (e.g., Datacolor Spyder, X-Rite i1Display) along with calibration software to precisely calibrate the monitor's brightness, contrast, color temperature, gamma value, and color space. This is the most accurate method and is recommended for professionals. Hardware calibration is the gold standard.
- **Software Calibration**: Use the operating system's built-in calibration tools (e.g., Windows "Color Calibration" or macOS "Display Calibrator Assistant") for basic calibration. While less precise than hardware calibration, it is far better than no calibration at all. Suitable for users on a limited budget.

**Calibration Frequency**: It is recommended to calibrate every 1-2 months, as a monitor's color performance drifts over time. Monitors age and require regular maintenance.

#### 15.3 Soft Proofing: Previewing Print Output on Screen

The Soft Proofing feature allows you to simulate, on your screen, how the final print will look on specific paper with specific inks.

- **Operation**: In Photoshop, go to `View > Proof Setup > Custom`.
- **Settings**: In the dialog, select an ICC profile that matches your print shop (usually provided by the printer). Enable "Simulate Paper Color" and "Simulate Black Ink" to more realistically simulate the paper's white point and the ink's black point.
- **Effect**: Once soft proofing is enabled, the on-screen image colors will change to simulate the printed result. You can then adjust the image accordingly to ensure the print matches your expectations.

### Chapter 16: File Organization and Naming Conventions

#### 16.1 Project Folder Structure: Organization is the Foundation of Efficiency

A chaotic project folder is a productivity killer. Establishing a clear, consistent folder structure allows you to quickly find any file you need at any time.

Good file organization is the foundation of a professional workflow. It saves time, reduces errors, and facilitates collaboration.

**Recommended Folder Structure Template**:

```
ProjectName_Date/
│
├── 01_Source_Materials/
│   ├── RAW/
│   ├── JPG/
│   └── Other_Assets/
│
├── 02_Working_Files/
│   ├── PSD/
│   ├── TIFF/
│   └── Backups/
│
├── 03_Output_Files/
│   ├── Web/
│   ├── Print/
│   └── Social_Media/
│
├── 04_References/
│   └── Inspiration/
│
└── 05_Project_Notes.txt
```

- **Source Materials**: Store all unprocessed original files. Never make any modifications here. This is your "raw data," the foundation of your work.
- **Working Files**: Store all Photoshop source files (PSD) and intermediate files (TIFF). Back up this folder regularly. Working files are your core assets.
- **Output Files**: Store all final exported files, organized by purpose (Web, Print, Social Media) into subfolders. Categorized storage makes management easy.
- **References**: Store reference images, sources of inspiration, etc. Reference materials help maintain creative direction.
- **Project Notes**: Record project requirements, client feedback, key steps, and other information. Document your workflow.

#### 16.2 File Naming Conventions: Let Files "Speak" for Themselves

A good file name should contain enough information so that you can understand the file's content and version even without looking at a thumbnail.

Good naming conventions make file management simple.

**Naming Convention Example**:
`ProjectName_Description_VersionNumber_Date.extension`

- **Example**: `ProductPoster_Final_v3_20250312.psd`
- **Breakdown**:
  - `ProductPoster`: Project name
  - `Final`: Content description
  - `v3`: Version number, facilitating revision history tracking
  - `20250312`: Date (YMD format for easy sorting)
  - `.psd`: File format

**Version Control**: Use version numbers (e.g., v1, v2, v3) or date suffixes to distinguish different versions. Avoid ambiguous names like "Final," "Final2," etc.

### Chapter 17: Automation & Scripts: Eliminating Repetitive Tasks

#### 17.1 Advanced Action Features: Conditions, Modal Controls, and Stops

Beyond simple recording and playback, Photoshop Actions support more advanced controls for smarter automation.

- **Insert Conditional**: Insert a "Conditional" into an Action, allowing it to execute different steps based on specific circumstances. For example, if the image is portrait orientation, execute step A; if landscape, execute step B. Conditional branching makes Actions smarter.
- **Insert Modal Control**: Insert "Modal Control" into an Action to pause playback at a certain step, allowing you to manually enter parameters (e.g., type a filename, adjust a slider) before continuing. Modal control adds flexibility.
- **Insert Stop**: Insert a "Stop" into an Action to display a dialog with custom instruction text when playback reaches that step. Stops can be useful for reminders or training.

#### 17.2 Scripts: More Powerful Automation

Scripts are an even more powerful automation tool than Actions. They allow you to use programming languages like JavaScript, AppleScript, and VBScript to control almost all of Photoshop's features, implementing complex automation tasks.

Scripts can accomplish complex tasks that Actions cannot.

**Common Script Applications**:
- **Batch Renaming**: Rename all files in a folder according to specific rules. Automates file management.
- **Batch Image Resizing**: Uniformly resize a batch of images to specific dimensions and add watermarks. Standardizes output.
- **Automated Contact Sheet Generation**: Automatically lay out a group of images onto a single large sheet to create an image index. Convenient for previewing.
- **Data-Driven Graphics**: Use data from a CSV file to batch-generate personalized images (e.g., certificates, business cards). Enables personalized mass production.

**Obtaining and Using Scripts**: You can find a large number of free Photoshop scripts on the Adobe Exchange website or GitHub. After downloading, place the script file (.jsx) into Photoshop's Scripts folder, then run it via `File > Scripts > Browse`.

### Chapter 18: Output and Delivery — The Final Touch

#### 18.1 Web Output: Optimizing for Screen

When outputting images for the web, you need to balance file size with image quality.

Web output requires balancing quality and file size.

**Save for Web**:
1. Go to `File > Export > Save for Web (Legacy)` (or use `File > Export > Export As`).
2. **Format Selection**:
   - **JPG**: Suitable for photos and images with rich gradients. Adjust the "Quality" slider to balance file size and quality. Typically, a quality setting of 60-80 is sufficient for web use. JPG is the most common web format.
   - **PNG-8**: Suitable for graphics with few colors, logos, and icons. Supports transparency and has a smaller file size. Ideal for icons and simple graphics.
   - **PNG-24**: Supports full color and transparency but results in larger files. Suitable for complex images that need transparency.
   - **GIF**: Suitable for simple animations or graphics with very few colors. GIF supports animation but has a limited color palette.
   - **WebP**: A next-generation image format that produces significantly smaller file sizes than JPG and PNG at equivalent quality. However, browser compatibility is not as widespread as traditional formats. Suitable for modern web development.
3. **Image Size**: Ensure the image width does not exceed the target platform's maximum width (e.g., 1080 pixels for Instagram). Check "Convert to sRGB" to ensure color consistency.
4. **Metadata**: Usually set to "None" to reduce file size, unless copyright information needs to be preserved.

#### 18.2 Print Output: Preparing for Paper

When preparing files for print, precision and quality are the primary considerations.

Print output requires high resolution and the correct color mode.

**Key Settings**:
1. **Resolution**: Ensure the image resolution is at least 300 DPI (dots per inch). For large-format printing (e.g., posters, banners), you can reduce it to 150-200 DPI.
2. **Color Mode**: Convert to CMYK color mode. Use `Edit > Convert to Profile` and select the ICC profile provided by your print shop. CMYK is the print standard.
3. **File Format**: Typically use TIFF or PDF format. TIFF supports lossless compression and layers; PDF is easy to transmit and preview.
4. **Bleed**: If the design needs to print to the edge of the paper (requiring bleed), you must reserve a bleed area in the design (usually 3mm per side) and include the bleed when exporting.
5. **Embed Fonts**: If using special fonts, select "Embed All Fonts" when exporting as PDF, or rasterize the text layers directly.

#### 18.3 Social Media Output: Optimizing for Each Platform

Different social media platforms have varying requirements for image dimensions, formats, and file sizes. Optimizing images for each platform ensures the best display quality.

Each social media platform has its own requirements and needs targeted optimization.

**General Recommendations**:
- **Dimensions**: Check each platform's latest image size requirements (e.g., Instagram square 1080x1080, Stories 1080x1920).
- **Format**: Generally use JPG format.
- **Color Space**: Always convert to sRGB.
- **Sharpening**: Images destined for web can benefit from a slight sharpening increase to compensate for compression-induced blur.

**Automation Tools**: Use Photoshop's Batch Processing feature to create Actions targeting different social media platforms, generating all required image sizes with a single click.
